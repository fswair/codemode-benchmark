--- Snippet 1 [normal]: Happy path: simple substitution (kitten -> sitting) ---
Code: levenshtein('kitten', 'sitting')
Output: 3

--- Snippet 2 [normal]: Identical strings should return 0 ---
Code: levenshtein('hello', 'hello')
Output: 0

--- Snippet 3 [normal]: Both empty strings should return 0 ---
Code: levenshtein('', '')
Output: 0

--- Snippet 4 [normal]: First string empty, second non-empty (all insertions) ---

--- Snippet 5 [normal]: Second string empty, first non-empty (all deletions) ---
Code: levenshtein('abc', '')
Output: 3

--- Snippet 6 [normal]: Single character same ---

--- Snippet 11 [normal]: Completely different strings of same length ---

--- Snippet 12 [normal]: Symmetry: distance(s,t) == distance(t,s) ---
Code: levenshtein('kitten', 'sitting') == levenshtein('sitting', 'kitten')
Output: True

--- Snippet 13 [normal]: Triangle inequality check ---
Code: d_ab = levenshtein('abc', 'axc')
d_bc = levenshtein('axc', 'axy')
d_ac = levenshtein('abc', 'axy')
d_ac <= d_ab + d_bc
Output: True

--- Snippet 14 [normal]: Distance equals length of longer string when one is empty ---
