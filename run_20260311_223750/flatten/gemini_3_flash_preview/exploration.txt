--- Snippet 1 [normal]: A simple flat list should return an identical new list. ---
Code: flatten([1, 2, 3])
Output: [1, 2, 3]

--- Snippet 2 [normal]: An empty list should return an empty list. ---
Code: flatten([])
Output: []

--- Snippet 3 [normal]: A basic 2D nested list is flattened. ---
Code: flatten([[1, 2], [3, 4]])
Output: [1, 2, 3, 4]

--- Snippet 4 [normal]: Deeply nested lists (multiple levels) are flattened into a single level. ---
Code: flatten([1, [2, [3, [4]]]])
Output: [1, 2, 3, 4]

--- Snippet 5 [normal]: Empty lists inside the main list should be removed during flattening. ---
Code: flatten([1, [], 2])
Output: [1, 2]

--- Snippet 6 [normal]: Deeply nested empty lists result in an empty list. ---
Code: flatten([[[]]])
Output: []

--- Snippet 7 [normal]: Flattening handles lists containing strings. ---
Code: flatten(["a", ["b", "c"], "d"])
Output: ['a', 'b', 'c', 'd']

--- Snippet 8 [normal]: Flattening handles lists containing booleans. ---
Code: flatten([True, [False, True]])
Output: [True, False, True]

--- Snippet 9 [normal]: None values are preserved as elements during flattening. ---
Code: flatten([None, [None]])
Output: [None, None]

--- Snippet 10 [normal]: Tuples are treated as non-list elements and kept as-is. ---
Code: flatten([1, (2, 3), [4, 5]])
Output: [1, (2, 3), 4, 5]

--- Snippet 11 [normal]: Dictionaries are treated as non-list elements and kept as-is. ---
Code: flatten([{"a": 1}, [{"b": 2}]])
Output: [{'a': 1}, {'b': 2}]

--- Snippet 12 [normal]: Check for idempotency; calling the function twice or comparing shouldn't change state. ---
Code: flatten([1, 2, [3, 4]]) == flatten([1, 2, [3, 4]])
Output: True

--- Snippet 13 [normal]: The returned list is a new object that can be mutated independently. ---
Code: res = flatten([1, [2, 3]]); res.append(4); res
Output: [1, 2, 3, 4]

--- Snippet 14 [normal]: Handling multiple references to the same list object. ---
Code: nested = [1, 2]; flatten([nested, nested])
Output: [1, 2, 1, 2]

--- Snippet 15 [normal]: Highly segmented nested lists are consolidated. ---
Code: flatten([[[1]], [[2]], [[3]]])
Output: [1, 2, 3]
