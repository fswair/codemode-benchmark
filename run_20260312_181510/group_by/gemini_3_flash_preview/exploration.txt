--- Snippet 1 [normal]: Happy path: grouping by an integer key present in all dicts ---
Code: group_by([{"id": 1, "name": "A"}, {"id": 2, "name": "B"}, {"id": 1, "name": "C"}], "id")
Output: {1: [{'id': 1, 'name': 'A'}, {'id': 1, 'name': 'C'}], 2: [{'id': 2, 'name': 'B'}]}

--- Snippet 2 [normal]: Boundary case: empty list of items returns an empty dict ---
Code: group_by([], "any_key")
Output: {}

--- Snippet 3 [normal]: Missing key: one dict is missing the specified key, should go under None. ---
Code: group_by([{"type": "fruit", "name": "apple"}, {"type": "vibe"}], "type")
Output: {'fruit': [{'type': 'fruit', 'name': 'apple'}], 'vibe': [{'type': 'vibe'}]}

--- Snippet 4 [normal]: Single group: all dicts have the same value for the key. ---

--- Snippet 5 [normal]: All missing keys: all dicts go under the None group. ---
Code: group_by([{"a": 1}, {"b": 2}], "c")
