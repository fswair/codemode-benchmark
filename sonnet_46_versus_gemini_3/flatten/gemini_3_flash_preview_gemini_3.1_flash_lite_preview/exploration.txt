--- Snippet 1 [normal]: Empty list input ---
Code: flatten([])
Output: []

--- Snippet 2 [normal]: Flat list input ---
Code: flatten([1, 2, 3])
Output: [1, 2, 3]

--- Snippet 3 [normal]: Simple nested list input ---
Code: flatten([1, [2, 3], 4])
Output: [1, 2, 3, 4]

--- Snippet 4 [normal]: Deeply nested list input ---
Code: flatten([1, [2, [3, 4], 5], 6])
Output: [1, 2, 3, 4, 5, 6]

--- Snippet 5 [normal]: Nested empty lists input ---
Code: flatten([[], [[]], [[[]]]])
Output: []

--- Snippet 6 [normal]: List with mixed types input ---
Code: flatten([1, "a", True, None])
Output: [1, 'a', True, None]

--- Snippet 7 [normal]: List of singleton lists input ---
Code: flatten([[1], [2], [3]])
Output: [1, 2, 3]

--- Snippet 8 [normal]: Linear nesting input ---
Code: flatten([1, [2, [3, [4, [5]]]]])
Output: [1, 2, 3, 4, 5]

--- Snippet 9 [normal]: Multiple branches of nesting input ---
Code: flatten([1, [2, 3], [4, [5, 6]]])
Output: [1, 2, 3, 4, 5, 6]

--- Snippet 10 [normal]: Very deeply nested singleton input ---
Code: flatten([[[[1]]]])
Output: [1]

--- Snippet 11 [normal]: Mixed nested lists and empty lists input ---
Code: flatten([1, [], 2, [3, []], 4])
Output: [1, 2, 3, 4]

--- Snippet 12 [normal]: Nested boolean input ---
Code: flatten([True, [False, [True]]])
Output: [True, False, True]

--- Snippet 13 [normal]: Nested float input ---
Code: flatten([1.1, [2.2, [3.3]]])
Output: [1.1, 2.2, 3.3]

--- Snippet 14 [normal]: Identity for empty list ---
Code: flatten([]) == []
Output: True

--- Snippet 15 [normal]: Idempotency check (flattening a flat list is identity) ---
Code: flatten([1, [2, 3]]) == flatten(flatten([1, [2, 3]]))
Output: True

Code: flatten([[], 1, [2, []], [3, [4, []]]])

--- Snippet 20 [normal]: Non-linear tree structure with balanced depth. ---
Code: flatten([[[1, 2], 3], [4, [5, 6]]])
Output: [1, 2, 3, 4, 5, 6]

--- Snippet 21 [normal]: Mixing singleton list wrappers with flat and nested elements. ---
Code: flatten([[1], [2], [[3], 4], 5])
Output: [1, 2, 3, 4, 5]

--- Snippet 22 [normal]: Very deep empty list structure with a terminal flat element. ---
Code: flatten([[[[[]]]], 1])
Output: [1]

--- Snippet 23 [normal]: Varying sub-list lengths and depths. ---
Code: flatten([[1, 2, 3], [], [4, [5, 6, [7, 8]]]])
Output: [1, 2, 3, 4, 5, 6, 7, 8]

--- Snippet 24 [normal]: Input lists generated by dynamic list creation. ---
Code: flatten([list(range(5)), [list(range(2))]])

--- Snippet 25 [normal]: Extremely deep nesting (8 levels) of a single element. ---
Code: flatten([[[[[[[[1]]]]]]]])

--- Snippet 26 [normal]: Boolean values nested at varying depths. ---
Code: flatten([[True], False, [True, [False]]])
Output: [True, False, True, False]

--- Snippet 27 [normal]: Alternating structure with single elements at the same nesting level. ---
Code: flatten([1, [2], 3, [4], 5])

Code: flatten([[1.5, 2.5], [3.5, [4.5, 5.5]]])
